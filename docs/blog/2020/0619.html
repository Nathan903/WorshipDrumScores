经文研读 |以赛亚书





以赛亚书 40:12-31

神是创造之主，唯独神有智慧


赛40:12 谁曾用手心量诸水、用手虎口量苍天、用升斗盛大地的尘土、用秤称山岭、用天平平冈陵呢。 

赛40:13 谁曾测度耶和华的心、〔或作谁曾指示耶和华的灵〕、或作他的谋士指教他呢。 

赛40:14 他与谁商议、谁教导他、谁将公平的路指示他、又将知识教训他、将通达的道指教他呢。 











上帝是无可比的，人与万物何其渺小


赛40:15 看哪、万民都像水桶的一滴、又算如天平上的微尘．他举起众海岛、好像极微之物。 

赛40:16 利巴嫩的树林不够当柴烧、其中的走兽也不够作燔祭。 

赛40:17 万民在他面前好像虚无、被他看为不及虚无、乃为虚空。 

赛40:18 你们究竟将谁比神、用甚么形像与神比较呢。 

赛40:19 偶像是匠人铸造、银匠用金包裹、为他铸造银鍊。

赛40:20 穷乏献不起这样供物的、就拣选不能朽坏的树木、为自己寻找巧匠、立起不能摇动的偶像。 













四个反问句，神才是公义的君王


赛40:21 你们岂不曾知道么、你们岂不曾听见么、从起初岂没有人告诉你们么、自从立地的根基、你们岂没有明白么。 

赛40:22 神坐在地球大圈之上、地上的居民好像蝗虫．他铺张穹苍如幔子、展开诸天如可住的帐棚。 

赛40:23 他使君王归于虚无、使地上的审判官成为虚空。 

赛40:24 他们是刚纔栽上、〔刚纔或作不曾下同〕刚纔种上、根也刚纔扎在地里、他一吹在其上、便都枯干、旋风将他们吹去、像碎秸一样。 











你不能不知

神掌管未来与现在

一切世上的东西都将成为虚空


赛40:25 那圣者说、你们将谁比我、叫他与我相等呢。 

赛40:26 你们向上举目、看谁创造这万象、按数目领出、他一一称其名．因他的权能、又因他的大能大力、连一个都不缺。 

赛40:27 雅各阿、你为何说、我的道路向耶和华隐藏．以色列阿、你为何言、我的冤屈　神并不查问。











神的大能大力从未停止


赛40:28 你岂不曾知道么、你岂不曾听见么、永在的　神耶和华、创造地极的主、并不疲乏、也不困倦。他的智慧无法测度。 

赛40:29 疲乏的、他赐能力。软弱的、他加力量。

赛40:30 就是少年人也要疲乏困倦、强壮的也必全然跌倒。 

赛40:31 但那等候耶和华的、必从新得力、他

们必如鹰展翅上腾、他们奔跑却不困倦、行走却不疲乏。 



一个无须回答的问题

赛40:12-14


1. 神是不是很伟大？神当然很伟大，这是一个无须回答的问题。无须回答而还要问，而且是一问再问不停的问，从不同的角度问，用不同的词句问，越问越显出神的不凡（「圣」的原意），越显出神的崇高。


2. 谁能够将神测透呢？「谁曾用手心量诸水、用手虎口量苍天、用升斗盛大地的尘土、用秤称山岭、用天平平冈陵呢。 谁曾测度耶和华的心、或作他的谋士指教他呢。」


3. 谁能够在神面前有份量呢？「看哪、万民都像水桶的一滴、又算如天平上的微尘．他举起众海岛、好像极微之物。」


4. 谁能够与神同等呢？「那圣者说、你们将谁比我、叫他与我相等呢。你们向上举目、看谁创造这万象、按数目领出、他一一称其名．因他的权能、又因他的大能大力、连一个都不缺。」


5. 谁能够向神隐藏呢？「雅各阿、你为何说、我的道路向耶和华隐藏．以色列阿、你为何言、我的冤屈　神并不查问。你岂不曾知道么、你岂不曾听见么、永在的神耶和华、创造地极的主、并不疲乏、也不困倦．他的智慧无法测度。」


地球的根基在哪里？


“神坐在地球大圈之上”，而地上的居民就如蝗虫一般。我们要知道，神的大能大力从未停止。神是不疲乏、不困倦的，他的智慧无法测度。可少年人再怎么强壮，都还是会累。


NO.2

坚信礼的反省问题


- 你相信你是个罪人吗？

我相信并确实知道自己是个罪人。


- 你怎么知道的？

从圣经、十诫我明白上帝的旨意，也看清楚我们不能达到上帝的标准。


- 你有为你的罪忧伤难过吗？

有，我因罪得罪了别人，也得罪了创造我们的上帝。




- 因我们的罪，我们有什么后果？

罪的工价是死。如果我们没有救恩，将会永远沉沦。


- 你盼望得救吗？

我望罪得赦免，并且永生。


- 你靠谁使你得救？

耶稣基督，我的救主，也是我生命的主。




- 耶稣如何救你？

他就我们，自己甘心走上十字架的道路，为我而死，赦我罪孽，救我离开永刑。


- 耶稣是谁？

是基督，是上帝的独生子，真人和真神。


- 你信仰基本撮要在哪里？

使徒信经，就是我们信仰的内容。




- 你相信主耶稣了，如何清楚得救？

-受洗，相信耶稣设立的洗礼能洗净我的罪。

-口里承认，心里相信。

-求主帮助，离开罪恶，追求圣洁。


- 你当如何坚固自己的属灵生命？


-守圣餐，并继续追求认识主。

-仰赖圣灵帮助，随着圣灵引导而活。


- 什么是教会，教会生活帮助你什么？

教会生活主要有崇拜，团契，和主日学查经，能帮我在灵命上成长。
