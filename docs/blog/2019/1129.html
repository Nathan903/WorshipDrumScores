1. 何謂同性戀？


所謂同性戀，是指有同性性傾向，並對同性有情感依賴，以及對同性有性感受，例如：性衝動、性幻想、手淫、性行為等。若每週發生次數頻繁，長達六個月以上者，則可界定為同性戀者。

 

2. 情感依賴


喜歡 --> 欣賞 --> 吸引--> 愛慕 --> 戀愛



*正常的同性關係應該在欣賞后就 停止。

4. 關於性別聖經怎麼說？



上帝創造人類，讓 一男一女 結為夫婦，並只讓人在婚姻關係中才可以有性行為。


創1:27-28 神就照著自己的形像造人，乃是照著他的形像造男造女。神就賜福給他們，又對他們說：要生養眾多，遍滿地面，治理這地，也要管理海裡的魚、空中的鳥，和地上各樣行動的活物。

利18:22 不可與男人苟合，像與女人一樣，這本是可憎惡的。


5. 如何對待同性戀者


聖經雖然譴責同性戀行為，卻不贊成人 憎恨 同性戀的人。聖經倒吩咐基督徒要「尊重各種各樣的人」。


彼前2:17 務要尊敬眾人，親愛教中的弟兄，敬畏神，尊敬君王。


同性戀的成因


1. 性別角色認同偏差

2. 不愉快的性經驗

3. 原生家庭的影響

4. 環境或媒體影響（色情文化）

5. 生理性的因素？


*聖經指出所有人生來就有 違反 上帝誡命的傾向，卻沒有說人是因為受到生理因素影響而有同性戀傾向。

有同性戀傾向的人怎麼辦？



聖經說：「要治死你們在地上的肢體，根絕淫亂、不潔、肉慾、有害的慾望」。（歌羅西書3:5）錯誤的慾望可以導致錯誤的行為，想要治死錯誤的慾望，就得控制自己的思想才行。

    

人要是讓思想充滿有益、美善的事，就會比較容易抗拒錯誤的慾望。對於有同性戀傾向的人來說，這樣做起初要經歷很大的掙扎，但只要不放棄，慢慢就會比較容易抗拒錯誤的慾望。上帝應許會幫助他們更新「思想背後的動力」。

4. 轉介至專業輔導機構
